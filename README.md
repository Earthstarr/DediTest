# DediTest

Unreal Engine 5.7 기반 1대1 FPS 팀 대결 게임 (서버-클라이언트 테스트 프로젝트)

## 개요

멀티플레이어 FPS 메커닉과 네트워크 리플리케이션을 테스트하기 위한 프로젝트입니다.

## 주요 기능

- **FPS 전투 시스템**: 조준, 사격, 재장전
- **이동 시스템**: 달리기, 스태미나 관리
- **네트워크 리플리케이션**: 서버-클라이언트 동기화
- **멀티플레이어 지원**: 1대1 팀 기반 대전

## 기술 스택

- Unreal Engine 5.7
- C++ & Blueprint
- Gameplay Ability System (GAS)
- Dedicated Server

## 서버/클라이언트 실행

**서버 빌드:**
- Server 타겟으로 패키징

**클라이언트 접속:**
- 콘솔: `open <서버IP주소>`
